# Scripts del modelo de crédito

- `make_dataset.py`: stub opcional para compatibilidad con el proyecto guía.
- `train.py`: entrena el modelo sobre `data/raw/Score.csv`.
- `evaluate.py`: evalúa el modelo usando `data/processed/credit_val.csv`.
- `predict.py`: genera `data/scores/final_score.csv` usando `data/processed/credit_score.csv`.
